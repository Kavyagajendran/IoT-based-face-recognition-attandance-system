## Project Description

The **IoT-Based Face Recognition Attendance System** is an automated attendance management solution that uses **computer vision** and **IoT technologies** to identify individuals through facial recognition and record attendance in real time.  
The system eliminates manual attendance processes, reduces human error, and ensures secure and accurate identification.

A camera captures live video frames, which are processed using face detection and recognition algorithms. Once a registered face is recognized, attendance is marked automatically and stored digitally. The IoT architecture enables centralized data storage and remote monitoring.

This project demonstrates the practical integration of **AI-powered vision**, **embedded systems**, and **networked data management**.

## System Architecture

1. Camera captures live video feed  
2. Face detection algorithm identifies faces  
3. Face recognition verifies registered users  
4. Attendance is logged with date and time  
5. Data is stored digitally and accessed via IoT framework  

---

## Working Principle

- The camera continuously monitors the entry area
- Faces are detected and compared with trained datasets
- Recognized faces trigger automatic attendance marking
- Attendance records are stored securely in digital format
- Data can be accessed remotely through IoT connectivity
